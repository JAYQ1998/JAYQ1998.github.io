# 1.需求说明

## 项目背景

随着互联网在中国的普及，网络招聘景气不断攀升。它允许以更加灵活的交互方式，提供更加丰富

的信息资源。为求职者提供更加多样、精确、新鲜的招聘信息。

## 项目意义

面对眼花缭乱的招聘信息，帮助求职者能够通过招聘信息的数据可视化，分析招聘薪资等信息的分

布，明确自己的方向，找到适合的工作。

## 项目目标

1. 实现通过搜索对相关招聘的主要信息进行展示、下载（选做）。

2. 对搜索出的信息进行地域、薪资、工作经验、学历、职责与要求等个方面的数据可视化分析。
